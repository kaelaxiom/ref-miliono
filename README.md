# ref-miliono

**Catálogo exhaustivo de modelos de negocio legales para generar el equivalente a $1M USD/año.**

> Generado autónomamente por agentes de investigación — 2 categorías / día. Crece solo.

---

## 📋 ¿Qué es esto?

Una base de conocimiento viva, generada por IA autónoma, que documenta **~40 modelos de negocio** legales y verificables capaces de generar $1M USD/año. Cada entrada incluye:

- 📊 **Matemáticas completas:** cuántos clientes a qué precio
- 🧭 **Camino de 0 a $1M:** fases, milestones, tiempos realistas
- ✅ **Casos reales** con nombres y fuentes verificables
- ⚠️ **Riesgos y trampas comunes** documentadas
- 📚 **Recursos accionables:** libros, cursos, comunidades

## 🤖 Sistema Autónomo

Este repo se construye automáticamente:

- **Cada 2 horas** un agente de investigación elige el siguiente modelo pendiente
- Investiga con fuentes web verificables
- Escribe el contenido siguiendo el template estructurado
- Hace commit y push automático
- **Sin intervención humana** — el repo crece solo
